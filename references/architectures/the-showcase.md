# The Showcase

**Best for:** wedding-venue, home-builder, detailing, barber
**Personality:** bold, visual, confident, crafted
**Tier:** 2-4
**Section count:** 6-8
**Animation budget:** 5 scroll-driven effects max

## Emotional Arc
- **Arrival:** Awe — "Look at this work"
- **Building:** Desire — "I want this for myself"
- **Decision:** Action — "I need to book/call now"

## Section Order + Visual Weight

| # | Section | Recipe Category | Weight | Spacing |
|---|---------|----------------|--------|---------|
| 1 | Hero (full-bleed image) | heroes | HERO | hero |
| 2 | Gallery / portfolio | gallery | STANDARD | tight |
| 3 | Before/after or process | gallery | HERO | standard |
| 4 | Services | services | WHISPER | standard |
| 5 | Testimonials | testimonials | STANDARD | standard |
| 6 | CTA break | cta-breaks | STANDARD | generous |
| 7 | Featured work detail | gallery | HERO | standard |
| 8 | Footer + contact | footers | WHISPER | tight |

## Rhythm Pattern
HERO → s → HERO → w → s → s → HERO → w

## Required Sections
- Hero (full-viewport image or video background)
- Gallery (grid, masonry, or lightbox)
- Services (brief — the images do the selling)
- Reviews (photo-paired if possible)
- CTA break (book / call / schedule)
- Footer (contact info integrated)

## Optional Sections
- Before/after slider
- Process / how-it-works
- Pricing packages
- Team / artist spotlight
- Instagram feed embed
- Area served map

## Signature Characteristic
Multiple hero-weight visual sections create a photo-forward experience where imagery does 80% of the persuasion. Text is minimal and purposeful — captions, not paragraphs.
