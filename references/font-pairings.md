# Curated Font Pairings

Stop defaulting to the same 3 font combinations. These are tested, beautiful pairings from Google Fonts that work reliably across all browsers and load fast. Each pairing includes the exact Google Fonts import URL and CSS variables.

The font pairing sets the entire typographic personality of the site. It's the second biggest quality signal after color — and the most common way AI-built sites expose themselves (Inter + Roboto = dead giveaway).

---

## How to Use

1. Find the mood category that matches the Business Personality Profile
2. Pick a pairing from that category
3. Copy the Google Fonts `<link>` tag into `index.html`
4. Use the CSS variable values in your `:root` tokens
5. **Never reuse the same pairing on consecutive builds in the same niche**

## Elegant & Refined

### DM Serif Display + DM Sans
Best for: cosmetic dental, med spa, wedding venues, luxury wellness
```html
<link href="https://fonts.googleapis.com/css2?family=DM+Serif+Display:ital@0;1&family=DM+Sans:wght@400;500;600;700&display=swap" rel="stylesheet">
```
```css
--font-display: 'DM Serif Display', serif;
--font-body: 'DM Sans', sans-serif;
```
Character: Warm serif authority with clean modern body. The italic is gorgeous.

## Banned Fonts (never use as primary)

- **Inter** — the #1 AI tell. Acceptable only as a fallback, never as the chosen font.
- **Roboto** — Android system font. Screams "default."
- **Arial / Helvetica** — system defaults signal zero design effort.
- **Space Grotesk** — was trendy in 2023, now overexposed in AI output.
- **Poppins** — borderline overexposed. Use only with strong justification.

## Font Pairing Selection Rules

1. **Match the personality.** Calm + Formal businesses get Elegant & Refined. Energetic + Bold businesses get Bold & Confident. Don't fight the personality with a mismatched font.

2. **Contrast display and body.** The pairing works because the display and body fonts are different enough to create hierarchy but share enough DNA to feel cohesive. Serif display + sans body is the safest bet. Sans display + sans body works when the weights and widths contrast strongly.

3. **Rotate within the mood.** If you just built a dental site with DM Serif Display + DM Sans, the next dental site gets Cormorant Garamond + Outfit or Playfair Display + Source Sans 3. Same mood category, different pairing.

4. **The display font IS the personality.** The body font's job is to be readable and stay out of the way. Don't get creative with body fonts — get creative with display fonts.

5. **Test at hero size.** Before committing, mentally render the business name at 80pt in the display font. Does it look right? "PRECISION AUTO DETAILING" in Bebas Neue = bold. In Cormorant Garamond = wrong niche.
