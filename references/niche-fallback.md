# Niche Fallback Mapping

When a business niche doesn't have its own dedicated niche profile, map it to the closest supported niche. Use that niche's color palettes, typography, architecture preferences, and emotional arc as a starting point, then adjust for the specific business.

## How to Use This Map

1. Look up the unmapped niche in the left column
2. Read the mapped niche's full profile from `references/niches/`
3. Use the mapped niche's recommended architecture, palette family, and typography
4. Adjust copy voice, imagery references, and CTA language for the actual business
5. Override any mapped elements that feel wrong for the specific client

## When Nothing Maps Well

If a business truly doesn't fit any mapping above:
- Default to **The Editorial** architecture (universally professional)
- Use the **general-premium** inspiration file
- Pick a neutral palette from the warm or cool families
- Prioritize: hero, services, reviews, CTA, contact, footer
